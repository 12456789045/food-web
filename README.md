Zomato Ultra-Premium Clone 🍔🍕🥗

A fully interactive and responsive food delivery web application inspired by Zomato
 built using pure HTML, CSS, and JavaScript.

** This project includes:

Responsive modern UI
Restaurant categories
Interactive cart system
Payment gateway simulation
Favorites system
Dashboard analytics
Account creation
Local storage support
Dynamic menus
Checkout flow
